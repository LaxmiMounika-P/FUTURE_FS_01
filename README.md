# 🌐 FUTURE_FS_01 - Portfolio Project

## 📌 About the Project

This is my personal portfolio website built using React.js as part of the Future Interns Full Stack Internship.

It showcases my skills, projects, contact details, and social links.

---

## 🚀 Features

- Responsive portfolio website
- About section
- Projects showcase
- Contact form with popup message
- Social media links (GitHub, LinkedIn, Email)

---

## 🛠️ Tech Stack

- React.js
- HTML5
- CSS3
- JavaScript
- React Icons

---

## 📂 Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/LaxmiMounika-P/FUTURE_FS_01.git
```

### 2. Go to project folder

```bash
cd FUTURE_FS_01
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
npm start
```
